**I’m Chaeyoung Jin, a Frontend Developer who translates product intent into interfaces users can understand.**  
I see interfaces not as a simple arrangement of features, but as **the first language users encounter** when experiencing a service.  
By understanding both **user flows and product goals**, I build UI experiences that deliver what users need in a natural way.  
I also care about turning the thoughts and structures behind implementation into **shared team assets**, so the work can grow in a better direction together.
